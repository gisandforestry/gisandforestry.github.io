Το setup του υπολογιστή μου

Date Sat 28 February 2015 Category misc Tags setup / linux / manjaro / arch

Εδώ και αρκετό καιρό ψάχνω να βρώ ένα διαφορετικό λειτουργικό σύστημα πέρα από τα Windows. Οι λόγοι πολλοί και από που να αρχίσει κανείς.

Κατα καιρούς έχω χρησιμοποιείσει όλα τα διαθέσιμα γνωστά λειτουργικά συστήματα, από mac OS μέχρι την πιο γυμνή έκδοση Linux. Όμως εδώ και μερικούς μήνες έχει πέσει στην αντίληψη μου μια διανομή Linux λίγο διαφορετική από τις άλλες. Αυτό που την κάνει να διαφέρει από κάθε άλλο λειτουργικό σύστημα που κυκλοφορεί είναι το γεγονός πως κάθε φορά που βγαίνει μια νέα έκδοση του λειτουργικού συστήματος δεν χρειάζεται να κάνει κανείς απεγκαάσταση του παλιού συστήματος και να εγκαταστήσει τη νέα έκδοση. Η φιλοσοφία αυτή ονομάζεται rolling release και ο λόγος γίνεται για την διανομή Arch Linux.

Η διανομή αυτή είναι το απόλυτο linux, η πιο "παρθένα" έκδοση Linux που μπορεί να βρεί κανείς. Μετά την εγκατάσταση αυτής της διανομής δεν υπάρχει το γνωστό σε όλους μας περιβάλλον εργασίας με τα επίσης γνωστά παράθυρα, αλλά αυτό που βλέπει ο χρήστης είναι μια μαύρη οθόνη με άσπρα γράμματά. Αυτό όμως είναι που κάνει αυτή την διανομή τόσο ξεχωριστή. Σου δίνει την δυνατότητ να διαμορφώσεις εσύ ο ίδιος το περιβάλλον που θα δουλεύεις όπως εσύ το θέλεις και οι επιλογές είναι πραγματικά άπειρες.

Επειδή όμως το στήσιμο αυτής της διανομής είναι αρκετά χρονοβόρο και απαιτεί εξειδικευμένες γνώσεις της επιστήμης των υπολογιστών, επέλεξα να χρησιμοποιώ μια διανομή που βασίζεται πάνω στο Arch και αυτή είναι η διανομή Manjaro.
